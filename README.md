# crx-hotload

Chrome extension hot Modules Reloader

# Install

> npm install -g path-fixxer

[![NPM](https://nodei.co/npm/crx-hotload.png)](https://nodei.co/npm/crx-hotload/)

# Usage

Create new extension Project
`npm crx-hmr create`

Start HMR server
`npm crx-hmr start`
